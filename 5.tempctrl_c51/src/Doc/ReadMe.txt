LCD1602实验步骤：
1、使用短路帽将J3端子的2和3短接（即选择VCC电源为5V）。
2、将开发板J24端子的P40和A0、P55和A1、P44和A2的短路帽去掉，开发板J25端子的P32和SCK、P34和SI、P35和RCK的短路帽都去掉。
3、将1602按照正确方向插到开发板J17上面。
4、MINI USB连接线给开发板通电下载程序，下载软件中内部IRC时钟选择11.0592MHZ。
5、下载程序后，LCD屏上会依次显示内容：
1）在屏第一行显示：LCD1602 check ok。
2）在屏第二行显示：test = x 。（x是从0到9不断变化的数字）


注：如果LCD屏没有显示，可使用螺丝刀拨动开发板上可调电阻RV3调节显示灰度。
